ResponsesPath

Указывает абсолютный путь к директории, содержащей шаблоны ответов, которые будут получены при срабатывании соответствующих правил.

Путь

ICAPSettings > ResponsesPath

Атрибуты

Этот элемент не имеет атрибутов.

Значение

Значение по умолчанию: /opt/kaspersky/ScanEngine/icap_data/templates.

Пример

Ниже приведен пример этого элемента.

<ResponsesPath>/opt/kaspersky/ScanEngine/icap_data/templates</ResponsesPath>
