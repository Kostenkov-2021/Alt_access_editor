KSNSettings

Задает настройки Kaspersky Security Network (KSN).

Эта секция необязательна. Если она отсутствует в файле конфигурации, KSN не используется.

Используя KSN, вы выражаете согласие на передачу "Лаборатории Касперского" данных, которые описаны в файле About data provision.txt. Смотрите также подробную информацию о предоставлении данных.

Путь

KSNSettings

Атрибуты

Этот элемент не имеет атрибутов.

Вложенные элементы

Этот элемент является контейнером для следующих вложенных элементов:

  • UseKSN
  • ObjectCheckOnDemandTimeoutMs
  • CacheSizeKb

Пример

Ниже приведен пример элемента KSNSettings.

<KSNSettings>

<UseKSN>0</UseKSN>

<ObjectCheckOnDemandTimeoutMs>10000</ObjectCheckOnDemandTimeoutMs>

<CacheSizeKb>30720</CacheSizeKb>

</KSNSettings>
