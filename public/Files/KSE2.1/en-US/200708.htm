Pass

Specifies the encrypted password that is used for authenticating on a proxy server. The value is encrypted by the kav_encrypt utility.

Path

ProxySettings > Pass

Attributes

This element has no attributes.

Value

If the User parameter and the Pass parameter are empty, an anonymous proxy server is used.

Example

The following is an example of this element.

<Pass>8mSHXFix3uL+RP9oNeKDZQ==</Pass>
