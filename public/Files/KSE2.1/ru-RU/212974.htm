Использование Kaspersky Scan Engine в режиме ICAP с Oracle Solaris Vscan

Вы можете настроить Kaspersky Scan Engine для работы с Oracle Solaris Vscan.

Чтобы настроить Kaspersky Scan Engine для работы с Oracle Solaris Vscan с помощью Kaspersky Scan Engine GUI:

  1. Перейдите на страницу Settings > Service.
  2. В поле Response mode address укажите avscan.
  3. Сохраните изменения.

Чтобы настроить Kaspersky Scan Engine для работы с Oracle Solaris Vscan с помощью файла конфигурации режима ICAP:

  1. Откройте kavicapd.xml для редактирования.
  2. В элементе ICAPSettings > RespModeAddress укажите avscan.
  3. Сохраните и закройте kavicapd.xml.
  4. Перезапустите Kaspersky Scan Engine, выполнив следующую команду:

    service kavicapd restart
