Distribution kit contents (Linux)

This section describes the contents of the Kaspersky Scan Engine distribution kit for Linux systems.

Distribution kit types

Kaspersky Scan Engine is distributed in the following types of distribution kits:

  • As an RPM package and a set of additional files
  • As a DEB package and a set of additional files
  • As a .tar.gz archive

    This type of distribution kit can be used instead of the RPM or DEB package.
