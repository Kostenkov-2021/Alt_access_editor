AuthSettings

Настройки аутентификации по токену API.

Этот элемент необязателен.

Путь

ServerSettings > AuthSettings

Вложенные элементы

Этот элемент является контейнером для следующих вложенных элементов:

  • UseAccessToken
  • AuthRequestField
  • UseBearerAuthScheme
  • AccessTokens

Пример

Ниже приведен пример этого элемента.

<AuthSettings>

<UseAccessToken>0</UseAccessToken> <!--Enable or disable API-token authentication -->

<AuthRequestField>Authorization</AuthRequestField>

<UseBearerAuthScheme>1</UseBearerAuthScheme>

<AccessTokens>

<Token>

<Description>ScanEngine Clients. Full access scope</Description>

<Enabled>1</Enabled>

</Token>
