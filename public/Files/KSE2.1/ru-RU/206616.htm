FormatRecognizerSettings

Указывает настройки Format Recognizer.

Путь

FormatRecognizerSettings

Атрибуты

Этот элемент не имеет атрибутов.

Вложенные элементы

Этот элемент является контейнером для следующих вложенных элементов:

  • FormatsToSkipScanning

    Указывает, какие форматы файлов решение Kaspersky Scan Engine не должно сканировать в режиме ICAP.

    Чтобы выключить функцию пропуска форматов, удалите элемент FormatRecognizerSettings из файла конфигурации или оставьте список FormatsToSkipScanning пустым.

Пример

Ниже приведен пример элемента FormatRecognizerSettings.

<FormatRecognizerSettings>

<FormatsToSkipScanning>

<KAV_FF_GENERAL_TXT/>

<KAV_FF_GENERAL_CSV/>

<KAV_FF_AUDIO_WMA/>

</FormatsToSkipScanning>

</FormatRecognizerSettings>
