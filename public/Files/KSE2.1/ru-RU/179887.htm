Сообщения об ошибках подключения HTTP-клиента

При возникновении ошибки подключения HTTP-клиент выводит сообщение об ошибке на консоль.

  • Invalid response

    Соединение было прервано, когда клиент ждал ответа от сервера. После этой ошибки клиент завершает работу.

  • За Invalid response следует CONNECTION_CLOSED

    Соединение было прервано, когда клиент передавал файл для сканирования на сервер.

  • За Invalid response следует CLIENT_TIMEOUT

    Превышен тайм-аут клиента (определенный параметром -t).

  • HTTP_ERROR 408 Request Timeout

    Превышен тайм-аут обработки (определенный параметром SessionTimeout).
