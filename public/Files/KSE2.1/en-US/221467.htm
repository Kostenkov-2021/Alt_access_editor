Exporting the settings

You can export Kaspersky Scan Engine settings as a ZIP archive with a set of configuration files.

The archive contains the following configuration files:

  • klScanEngineUI.xml
  • For HTTP mode:
    • kavhttpd.xml
    • httpdkavlog.ini
  • For ICAP mode:
    • kavicapd.xml
    • icapdkavlog.conf

To export the settings,

Click the Export configuration files button on the bottom left of any Kaspersky Scan Engine setting page.
