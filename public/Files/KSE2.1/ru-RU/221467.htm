Экспорт настроек

Вы можете экспортировать параметры Kaspersky Scan Engine в виде ZIP-архива с набором файлов конфигурации.

В архиве находятся следующие файлы конфигурации:

  • klScanEngineUI.xml
  • Для режима HTTP:
    • kavhttpd.xml
    • httpdkavlog.ini
  • Для режима ICAP:
    • kavicapd.xml
    • icapdkavlog.conf

Чтобы экспортировать настройки,

Нажмите кнопкуExport configuration files в левом нижнем углу любой страницы параметров Kaspersky Scan Engine.
